AI CodeLint

Boost code quality as you type.
Catches logic bugs that survive compilation and traditional linting.

Heavy development. This site and the project overall are incomplete. Expect rough edges.

Install

macOS / Linux

curl -fsSL https://raw.githubusercontent.com/RussellGN/AI-CodeLint/main/scripts/install.sh | sh

Windows (PowerShell)

irm https://raw.githubusercontent.com/RussellGN/AI-CodeLint/main/scripts/install.ps1 | iex

Usage

  1. 1

    Install the binary using the scripts above.

  2. 2

    Get an API key from OpenRouter.

  3. 3

    Run the configuration wizard.

    ai-codelint --configure
  4. 4

    Lint a file from the CLI—

    ai-codelint --mode cli --path <path/to/file>

    or

    install the VS Code extension and open any code file in a new editor window.
